# 说明
本项目基于SpianlHDL和LiteX，基于VexRiscv构建具有浮点运算单元的CPU,通过LiteX构建外设并组成SOC。板子上电后从SD卡读取并运行程序，按键触发后，采集电压并进行快速傅里叶变换，再通过OLED显示出来。
采用Sipeed Tang Premier 20K(Gowin GW2A FPGA)+自制扩展板。
移植了U8g2。
